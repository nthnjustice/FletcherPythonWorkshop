Misc. Functions

  • print() # output

  • input() # take input as string

  • int() # convert to integer

  • str() # convert to string

Lists

myList = []

  • myList.append("Something") # add to the end of a list

  • myList.insert(3, "Add this") # pick index to insert in front of, pass value

  • myList.remove(7) # removes first instance of the value passed

  • myList.pop(5) # remove the element at the INDEX passed to the function

  • anIndex = myList.index("Something") # returns the index for the value passed

  • myList.count(5) # returns the number of times a given element appears

  • myList.sort() #no return value, sorts behind the scenes

  • length = len(myList) # returns the number of elements in the list

  • total = sum(myList) # returns the sum of a list of integers

Files

f = open("aFile.txt", "r") # name of file (WITH extension), r = read or w = write

  • f.readline() # reads the pointer's line and moves the pointer down

  • f.read() # returns the WHOLE file as a SINGLE string

  • f.write("A string") # writes text to the open document, mode must be "w"

  • f.close() # close the file

Strings

aString = "Hello World"

  • aString.upper() # converts all characters to uppercase

  • aString.lower() # converts all characters to lowercase

  • aString.split() # breaks the string into elements of a list based on the argument

Dictionaries

myDict = {}

  • myDict.keys() # returns a list of the dictionary keys

  • myDict.values() # returns a list of the dictionary values

  • del myDict["someKey"] # deletes the selected key and value

  • myDict.clear() # clear all key-value pairs in the dictionary

  • del myDict # delete the entire dictionary from memory
